# Cadastral Mapping & Topology Validation (Khasra Parcel Project)

**OVERVIEW**

This project demonstrates cadastral mapping using ArcGIS Pro, including georeferencing, digitization, topology validation, and layout creation.

---

**PROJECT DESCRIPTION**

Performed cadastral mapping using ArcGIS Pro, including georeferencing and digitization of parcel boundaries, rivers, water courses, and command areas. Conducted topology validation to identify and correct overlaps and gaps, ensuring spatial accuracy, and generated final layout maps.

---

**WORK DONE**

- Digitized cadastral parcel boundaries  
- Mapped additional features such as rivers, water courses, and command areas  
- Assigned Khasra numbers and attribute data  
- Created and validated topology rules  
- Identified and corrected spatial errors (overlaps & gaps)  
- Generated final layout map output  

---

**TOOLS USED**

- ArcGIS Pro  
- MS Excel (for attribute handling)

**ERROR IDENTIFICATION & CORRECTION**

Initial dataset contained topology errors such as overlaps and gaps between parcel boundaries. These errors were identified using topology rules in ArcGIS Pro and corrected using editing tools to ensure spatial accuracy and data quality.
