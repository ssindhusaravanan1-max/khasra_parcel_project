# Khasra Parcel Mapping Project

## Overview
Cadastral mapping project using ArcGIS Pro including georeferencing, digitization, topology validation, and layout creation.

## Tools Used
- ArcGIS Pro
- Geodatabase (.gdb)
- Topology tools
- Layout & Labeling

## Workflow
- Georeferencing base map
- Parcel digitization
- Geometry check & repair
- Topology validation (gaps/overlaps)
- Labeling Khasra numbers
- Final layout export (PDF)

## Output
- Clean cadastral map
- Error-free dataset
- Final GIS PDF map
